# 🏥 CarePlus Hospital Website

A responsive hospital booking landing page built using HTML & CSS.

## 🚀 Features
- Clean and modern UI
- Appointment booking form
- Embedded Google Maps location
- Audio welcome message
- Responsive design

## 🌐 Live Demo
https://Ayeshar17.github.io/careplus-hospital-website/

## 📁 Tech Stack
- HTML5
- CSS3

## 📌 Purpose
This project was built as a learning project to understand web development basics and deployment using GitHub Pages.
