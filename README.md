# 🏥 CarePlus Hospital Website

A clean and responsive hospital booking landing page designed to provide a smooth and simple user experience for patients.

---

## 🚀 Live Website
🔗 https://Ayeshar17.github.io/careplus-hospital-website/

---

## ✨ Features

- 🧾 Easy appointment booking form  
- 📍 Integrated Google Maps location (CARE Hospital, Banjara Hills)  
- 🔊 Welcome audio for user engagement  
- 🎨 Clean and modern UI design  
- 📱 Responsive layout for different screen sizes  

---

## 🛠️ Tech Stack

- HTML5  
- CSS3  

---

## 🎯 Project Purpose

This project was built to practice:
- Frontend development fundamentals  
- UI/UX structuring using semantic HTML  
- Styling with CSS  
- Deploying a live website using GitHub Pages
